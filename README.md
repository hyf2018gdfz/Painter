# Painter

大作业：一个所见即所得的画图软件。

# Feature

- 支持撤回（Ctrl + Z），前进（Ctrl + Y），删除（Delete）
- 选择模式，可点击图形或拖动鼠标选择矩形内的图形，可批量选择
- 放大缩小（Ctrl + 滚轮），旋转画布（Shift + 滚轮）
- 矩形模式，拖动鼠标绘制矩形，按 Shift 绘制正方形
- 椭圆模式，拖动鼠标绘制椭圆，按 Shift 绘制圆
- 多边形模式，左键单击绘制多边形节点，单击起点或双击结束绘制
- 直线模式，拖动鼠标绘制直线
- 自由曲线模式，拖动鼠标绘制自由曲线

# Experimental Feature

- 保存图片，会生成一个 `try.png`
